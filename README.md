🚀 End-to-End Test Automation Framework built with industry-standard tools.

This project demonstrates a scalable automation framework covering both UI and API testing using modern QA practices.

🔧 Tech Stack:
- Java
- Selenium WebDriver (UI Automation)
- Cucumber (BDD - Gherkin)
- TestNG (Test Execution)
- RestAssured (API Testing)
- Allure Reports (Test Reporting)
- Maven (Build & Dependency Management)

🧱 Framework Design:
- Page Object Model (POM)
- Cucumber Step Definitions
- Hooks & Driver Factory pattern
- Tag-based execution strategy (@smoke, @regression, @api)

🌐 Test Coverage:
- UI Login Automation (Selenium)
- API Login Testing (RestAssured)

📊 Reporting:
- Allure integrated for rich test reports with steps, logs, and results visualization

⚙️ CI/CD:
- Designed to run in Jenkins pipelines
- Maven-based execution (`mvn clean test`)
- Ready for GitHub integration

🎯 Goal:
Provide a clean, maintainable, and scalable automation framework aligned with real-world QA engineering practices.
